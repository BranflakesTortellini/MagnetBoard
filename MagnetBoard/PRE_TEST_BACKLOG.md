# Magnet Board — pre-test backlog

These are good items to consider after first device testing, not necessarily before it.

## Likely next refinements

1. Full multi-plan support: separate boards such as “Date night,” “Parents visit,” and “Weekend trip.”
2. Pretty itinerary view: a simplified day card that hides technical details.
3. More robust import/export/backup of the JSON data.
4. Better route optimization using MapKit travel times rather than straight-line distance.
5. Location result favorites / recently used places.
6. Better Dynamic Type tuning for larger text sizes.
7. VoiceOver audit.
8. A calmer visual theme option if the cute theme feels too busy.
9. A “focus mode” that hides advanced details and shows only Add, Board, Plan, Undo.
10. Optional passcode/Face ID is probably unnecessary unless sensitive calendar data is added.
